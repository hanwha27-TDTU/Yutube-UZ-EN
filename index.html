☁️ Supabase 클라우드 연동

Supabase 프로젝트의 URL과 anon key를 입력하면
문제은행이 자동으로 동기화됩니다.

Supabase 테이블 생성 SQL: 
create table if not exists questions (
  id text primary key,
  created_at timestamptz default now(),
  updated_at timestamptz default now(),
  deleted_at timestamptz,
  data jsonb not null
);

create table if not exists language_sync_meta (
  key text primary key,
  value text,
  updated_at timestamptz default now()
);
RLS를 켰다면 anon key가 읽기/쓰기 가능한 정책을 별도로 설정하세요.
