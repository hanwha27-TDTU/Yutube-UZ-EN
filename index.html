📁 데이터 관리

백업과 파일 입출력은 Supabase 동기화와 별개입니다. 클라우드 최종본을 바꾸려면 동기화 창의 기기→최종본 덮어쓰기를 사용하세요.

전체 백업

문제, stem, 해설, 카테고리, 난이도, 생성/수정 시간을 모두 보존하는 JSON 백업입니다.

앱 파일 백업

현재 문제은행을 이 HTML 앱 파일 안에 구워 넣어 다운로드합니다. 오프라인 보관용입니다.

TXT 가져오기

간단한 문제 입력용입니다. 가져온 문제는 Imported / Medium으로 추가됩니다.

TXT 내보내기

질문과 보기만 포함합니다. stem, 해설, 카테고리, 난이도는 빠집니다.

웹 앱

USMLE 문항을 직접 만들고 반복 학습할 수 있도록 설계한 개인 학습용 앱입니다. 의료 판단에는 원문 출처와 최신 임상지침을 별도로 확인해야 합니다.

업데이트 이력

최신 · 전수점검 버그 수정

  • 문자열/UUID 문제 id에서도 문제은행 수정·삭제·선택 기능이 깨지지 않도록 보강
  • 오프라인 삭제가 재연결 후 되살아나지 않도록 로컬 tombstone과 병합 동기화 추가
  • 전체 백업에 삭제 기록과 현재 파일명 정보를 함께 반영

v2.2 · 데이터 관리와 개발자 정보 UI 추가

  • 네비의 파일 백업 버튼을 데이터 관리 창으로 이동
  • 전체 JSON 백업 내보내기와 가져오기(교체/병합) 추가
  • Supabase 동기화 창에서 SQL 표시를 제거하고 운영 버튼 중심으로 정리
  • 개발자 정보 버튼을 우측 상단에 배치

v2.1 · Supabase 기기간 동기화 UI 정리

  • 클라우드 최종본, 기기→최종본 덮어쓰기, 클라우드→로컬 개념으로 문구 정리
  • revision과 canonical snapshot 기반 동기화 구조 적용
  • 연결 테스트 성공 시 클라우드 최종본을 먼저 불러오도록 변경

v2.0 · 보안과 데이터 정합성 보강

  • HTML 저장 마커 보강과 script 종료 문자열 이스케이프 처리
  • 보기/해설/문제은행 미리보기 HTML escape 적용
  • 정답 보기 공백 저장 방지와 Fisher-Yates 셔플 적용
